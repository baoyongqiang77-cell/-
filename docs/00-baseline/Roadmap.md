# 无人机智能巡检系统开发路线图

## 1. 总体里程碑

| 里程碑 | 目标 | 算法要求 | 验收输出 |
| --- | --- | --- | --- |
| M1 平台闭环可用 | 飞控、媒体入库、抽帧、模拟推理、告警、工单、标注、训练、发布流程跑通 | 可使用模拟算法服务，不要求真实模型指标达标 | U0-U5 单元测试文档、平台闭环演示记录、模拟算法回放结果 |
| M2 一期最终验收 | 14 个一期真实算法接入并通过指标、回放、人工抽检和双硬件适配要求 | 必须使用真实模型；模拟后端不能替代真实算法验收 | 一期 14 算法验收测试文档、模型评估报告、UAT 记录、系统级验收报告 |

## 2. 开发单元顺序

### U0 基础工程

依赖输入：
- 技术栈、部署环境、数据库、对象存储、消息队列。
- 平台方与客户租户初始化信息。

交付输出：
- 后端骨架、前端骨架、认证/RBAC、租户、功能授权、数据授权、审计、配置、CI/CD、迁移脚本、OpenAPI。

完成定义：
- 可登录、可鉴权、可按租户过滤数据、可按功能授权控制模块、可写审计、基础服务可一键启动。

验收重点：
- 平台方租户和客户租户类型正确。
- 客户租户访问训练/发布返回 `FEATURE_403`。
- 未授权客户数据进入训练返回 `DATA_GRANT_412`。

### U1 飞控平台

依赖输入：
- 大疆机场 3 适配层模拟器、设备台账、GIS 最小资产、网关网络策略。

交付输出：
- 设备/机巢、航线、任务、审批、遥测、任务状态机、大疆机场 3 适配网关。

完成定义：
- 模拟设备可完成创建、审批、下发、执行、异常、媒体同步全流程。
- 真实设备验收需大疆机场 3 设备、DJI 凭证、客户网络环境和网关出口白名单。

验收重点：
- 业务平台不直接调用 DJI API。
- 状态跳转合法。
- 内网业务服务不能绕过网关访问互联网。

### U2 媒体与分析主链路

依赖输入：
- 对象存储、消息队列、模拟算法服务、租户路径规则。

交付输出：
- 媒体入库、断点续传、抽帧、分析任务、推理队列、告警工单。

完成定义：
- 使用模拟算法完成端到端闭环。
- 失败和重试状态可验证。
- 媒体与抽帧产物按租户路径隔离。

验收重点：
- 弱网、重复回调、重复分片不丢文件、不重复入库。
- 告警到工单 P95 <= 10 秒。
- 跨租户查询返回 `TENANT_403` 或 `TENANT_404`。

### U3 推理运行时

依赖输入：
- 模型注册、算法主数据、NVIDIA 测试环境、国产测试环境或模拟后端。

交付输出：
- 统一 gRPC 协议、运行时适配器、模型制品管理、后端一致性检查。

完成定义：
- NVIDIA 与国产后端或模拟后端实现同一 `Predict` 契约。
- 真实硬件兼容测试必须使用真实 NVIDIA/国产卡。

验收重点：
- 输出 schema 一致。
- 后端不得静默切换。
- 国产硬件未锁定不得宣称硬件验收通过。

### U4 数据标注平台

依赖输入：
- `frame_assets`、算法标签体系、用户角色、客户租户功能授权。

交付输出：
- 样本池、标注任务、标注工具、初检/复检、数据集导出。

完成定义：
- 平台方可配置和质检标注任务。
- 客户租户可处理本租户授权标注任务。
- 样本来源可追踪。

验收重点：
- FROZEN 后不可修改。
- 客户租户不能处理其他租户样本。
- 未授权数据集导出返回 `FEATURE_403`。

### U5 训练与算法库

依赖输入：
- 数据集版本、训练环境、模型基座、`tenant_data_grants`。

交付输出：
- 训练任务、指标、模型版本、评估、灰度、发布、回滚、数据授权校验。

完成定义：
- 训练和模型发布默认仅平台方可用。
- 客户租户训练需单独授权。
- 未达标模型不能发布。

验收重点：
- 未授权客户数据不能进入训练。
- 发布和回滚有审计。
- 双硬件 runtime_artifacts 可记录和校验。

### U6 一期 14 算法接入

依赖输入：
- 一期算法矩阵、冻结验收集、训练结果、阈值策略。

交付输出：
- 14 个真实算法模型、阈值、输出 schema、验收报告。

完成定义：
- 每个算法通过离线指标、视频回放和人工复核抽检。
- 模拟算法不得计入通过。

验收重点：
- 冻结验收集 checksum。
- 每个算法有评估报告和回放记录。
- 任一一期算法未达标则 U6 不通过。

### U7 联调验收

依赖输入：
- U0-U6 全部完成。
- 国产卡型号、外部接口模式和网络分区策略已锁定。

交付输出：
- 性能、安全、弱网、断点续传、双硬件一致性、租户隔离、功能租赁授权、数据训练授权、网络边界、UAT 报告。

完成定义：
- 系统级流程、非功能指标、平台方全功能、客户租户受限功能、租户隔离、网络出口控制和客户验收场景均通过。

验收重点：
- 外部生产接口、硬件或网络边界未锁定时，不得进入生产验收。
- 租户隔离、功能授权、数据授权、网络边界任一项不通过，则 U7 不通过。

## 3. 推荐排期结构

| 阶段 | 工作包 | 主要产出 |
| --- | --- | --- |
| 第 1 阶段 | U0 基础工程 | 工程骨架、租户/RBAC/授权/审计、基础 API、OpenAPI |
| 第 2 阶段 | U1 + U2 主链路 | DJI 模拟器、飞控任务、媒体入库、抽帧、模拟分析、告警工单 |
| 第 3 阶段 | U3 + U4 | 推理 gRPC、双后端契约、标注平台、数据集封版 |
| 第 4 阶段 | U5 | 训练、评估、模型版本、发布、回滚、数据授权 |
| 第 5 阶段 | M1 验收 | 平台闭环演示、U0-U5 测试文档、模拟算法回放 |
| 第 6 阶段 | U6 | 14 个真实算法接入、离线评估、回放、人工抽检 |
| 第 7 阶段 | U7 + M2 | 系统联调、性能/安全/弱网/租户/网络/硬件验收、UAT |

## 4. 关键门禁

| 门禁 | 必须满足 |
| --- | --- |
| U0 进入 U1/U2 | 租户、权限、功能授权、数据授权、审计、幂等和异步任务中心可用 |
| U1 进入真实设备联调 | 适配网关模拟器通过；网关出口白名单策略明确；业务服务不能直连外网 |
| U2 进入 M1 | 媒体入库、抽帧、模拟推理、告警、工单、复核闭环可跑通 |
| U3 进入硬件验收 | NVIDIA 和国产后端接口契约一致；真实硬件型号、驱动、SDK、镜像已锁定 |
| U5 进入模型发布 | 模型评估达标；客户数据授权可追溯；发布和回滚可审计 |
| U6 进入 U7 | 14 个真实算法均达标；冻结验收集 checksum 和回放记录齐全 |
| U7 通过 | 性能、安全、弱网、租户隔离、功能授权、数据授权、网络边界和 UAT 均通过 |

## 5. 风险前置任务

| 风险 | 前置任务 |
| --- | --- |
| DJI API 权限/证书不可用 | 先完成适配层接口和模拟器；真实接入单独验收 |
| 统一身份协议未确认 | 本地账号先行，统一身份适配器接口预留 |
| OA/空域审批接口未确认 | 内置审批流模拟器先行，保留人工录入和 Webhook 适配 |
| GIS 坐标系未确认 | 最小资产台账先行，一期最低验收到 L1 资产级 |
| 对象存储接口未确认 | 开发使用 MinIO，生产前替换为客户认可内网存储 |
| 国产卡型号未确认 | 国产后端模拟器做契约测试，不做真实硬件通过声明 |
| 算法验收集未冻结 | U6 前冻结验收集，生成 checksum，冻结后不得调整样本和标签 |

## 6. M1 验收输出清单

- U0-U5 单元测试文档。
- 平台闭环演示记录。
- 大疆机场 3 适配层模拟器回放记录。
- 媒体入库、抽帧、模拟推理、告警、工单和复核记录。
- 标注、数据集、训练、评估、发布和回滚演示记录。
- 租户隔离、功能授权和数据授权基础测试记录。

## 7. M2 验收输出清单

- 一期 14 算法验收测试文档。
- 冻结验收集版本、checksum 和样本统计。
- 每个算法离线指标报告、视频回放结果、人工抽检记录。
- 模型评估报告、双硬件一致性报告。
- 性能、安全、弱网、断点续传报告。
- 租户隔离、功能租赁授权、数据训练授权报告。
- 网络边界、防火墙策略、出口白名单和阻断测试报告。
- UAT 记录和签字材料。
