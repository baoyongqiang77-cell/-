# U1 飞控平台功能测试文档

## 验收范围

- 大疆机场 3 适配层模拟器。
- `schema:flight_event_payload` 回调字段规范。
- 飞行任务状态机与非法跳转拒绝。
- 仅大疆机场 3 适配网关允许访问互联网的网络边界规则。

## 自动化用例

| 用例 | 结果 |
| --- | --- |
| 绑定、遥测、低电量、媒体上传完成均转换为统一 payload | PASS |

## 验收结论

U1 模拟器与状态机规则级验收通过。真实 DJI Cloud API、真实设备、证书、回调协议和客户网络出口白名单仍为待确认项，不能据此声明真实飞控生产验收通过。
