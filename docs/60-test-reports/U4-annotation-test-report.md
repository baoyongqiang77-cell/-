# U4 数据标注平台测试文档

## 验收范围

- 样本池与标注任务租户归属。
- 客户租户只能处理本租户授权样本。
- 标注提交、退回、复检通过流程。
- `FROZEN` 数据集不可修改。
- 客户租户导出数据集需单独开通 `DATASET_EXPORT`。

## 自动化用例

| 用例 | 结果 |
| --- | --- |
| 客户租户可处理本租户已分配样本 | PASS |
| 客户租户创建包含其他租户样本的任务返回 `TENANT_403` | PASS |
| 标注任务支持提交、退回、再次提交、最终通过 | PASS |
| `FROZEN` 数据集追加样本返回 `STATE_409` | PASS |
| 客户默认导出数据集返回 `FEATURE_403`，授权后可导出本租户路径 | PASS |

## 验收结论

U4 规则级验收通过。真实标注工具 UI、COCO/YOLO 导出包内容和质检统计报表仍需在后续完整应用中实现和验收。
